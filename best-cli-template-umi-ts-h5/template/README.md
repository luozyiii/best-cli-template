# 项目
> Umi + TS H5项目模板

## 使用

- 安装

```bash
$ yarn
```

- 启动服务

```bash
$ yarn start
```

## 基础示例
- 登录(admin/123456)
- 基本用法
- 基础表单
- 权限路由
```javascript
// 使用高级组件 wrappers
// router 中配置
wrappers: ['@/wrappers/authAdmin'],
```
